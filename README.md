# Prototipo Barrio Seguro

Prototipo interactivo de cuatro subfuncionalidades principales del sistema comunitario:

- Boton de panico
- Reporte de incidente
- Mapa de riesgo
- Validacion de identidad

## Como visualizarlo

Abra el archivo `prototipo_barrio_seguro_validacion.html` en cualquier navegador web.
